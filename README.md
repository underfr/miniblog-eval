
# Comment lancer la stack ?
Executer la commande

>     docker compose up -d

  

# PORTS

 - Application WEB

    

> `8080`

  

 - PHPMyAdmin

    

> `8081`

  

 - MailHog

    

> `8025`

  

 - MySQL

    

> `3306`

  

# Pour importer / Restaurer la base

    
Executer les commandes dans l'ordre:
> `docker compose down -v`

> `docker compose up -d`
